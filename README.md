# Do-It
Application to organize your tasks

## Documentation

Visit https://romach.github.io/Do-It/ to see actual documentation.

To open documentation locally:

Install `Docsify`:
```
npm i docsify-cli -g
```

Run `Docsify`:
```
docsify serve docs
```

Stop `Docsify`:
```
Ctrl+C
```
